Hawkes过程笔记

发表于 2018-09-29

基本概念

点过程

点过程定义
有一系列的非负随机变量$\left { Tk \right }{k=1,2,\cdots}$对所有$k$都有$Tk \leq T{k+1}$
Counting Process定义
$$N(t)=\sum{k=1,2,\cdots}\mathbb{1}{Tk \leq t}$$
密度函数
$$\lim{h \to 0^+} \frac{E(N(t,t+h])}{h}$$

泊松过程

指数分布

指数分布PDF
$$
f(x)=\left{\begin{matrix}
\lambda e^{-\lambda x}& if \quad x \geq 0\
0 & if \quad x < 0
\end{matrix}\right.
$$
指数分布CDF
$$
F(x)=\int_{-\infty }^{x}f(y)dy =\left{\begin{matrix}
1- e^{-\lambda x}& if \quad x \geq 0\
0 & if \quad x < 0
\end{matrix}\right.
$$
参数$\lambda$
$$E[X] = \frac{1}{\lambda}$$
泊松分布
$$P\left { X = n \right } = \frac{e^{-\lambda}\lambda^n}{n!}$$
参数$\lambda$
$$E[X] = \lambda$$

齐次泊松过程

定义

  1. $N(0)=0$
  2. 独立增量
  3. $P\left { N(t+s) - N(s) = n \right } = e^{-\lambda t}\frac{(\lambda t)^n}{n!}$

等价定义

  1. $N(0)=0$
  2. 独立增量
  3. $P\left { N(t+h) - N(t) = 1 \right } = \lambda h + o(h) $
  4. $P\left { N(t+h) - N(t) \geq 2 \right } = o(h) $

非齐次泊松过程

定义

  1. $N(0)=0$
  2. 独立增量
  3. $P\left { N(t+h) - N(t) = 1 \right } = \lambda(t) h + o(h) $
  4. $P\left { N(t+h) - N(t) \geq 2 \right } = o(h) $

自激点过程

点过程称为自激的如果其密度函数不仅和时间$t$有关，还和点过程过去的历史有关：
$$\lambda(t|\textit{F}{t^-}^N) = \lim{h \to 0^+} \frac{E[N(t+h)-N(t)|\textit{F}_{t^-}^N]}{h}$$

带有指数核的Hawkes过程

单变量Hawkes过程

定义

  1. $$N(0)=0$$
  2. $$\lambda(t) = \mu + \int{0}^{t}\alpha e^{-\beta(t-s)}dN(s)=\mu+\sum{k:t_k<t}\alpha e^{-\beta(t-s)}$$
  3. $$P\left { N(t+h) - N(t) = 1 \right } = \lambda(t) h + o(h) $$
  4. $$P\left { N(t+h) - N(t) \geq 2 \right } = o(h) $$
