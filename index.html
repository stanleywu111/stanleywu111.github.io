齐次泊松过程

定义

  1. $N(0)=0$
  2. 独立增量
  3. $P\left \{ N(t+s) - N(s) = n \right \} = e^{-\lambda t}\frac{(\lambda t)^n}{n!}$

等价定义

  1. $N(0)=0$
  2. 独立增量
  3. $P\left \{ N(t+h) - N(t) = 1 \right \} = \lambda h + o(h) $
  4. $P\left \{ N(t+h) - N(t) \geq 2 \right \} = o(h) $

非齐次泊松过程

定义

  1. $N(0)=0$
  2. 独立增量
  3. $P\left \{ N(t+h) - N(t) = 1 \right \} = \lambda(t) h + o(h) $
  4. $P\left \{ N(t+h) - N(t) \geq 2 \right \} = o(h) $
